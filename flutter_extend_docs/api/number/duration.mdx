Stop creating verbose Duration objects! These simple getters allow you to write time-based code in a highly readable, expressive way, speeding up everything from animation delays to timeout settings.

```dart num icon="hashtag" lines wrap
// Readable Delay
await Future.delayed(2.seconds);

// Clear Timeout
myFuture.timeout(5.minutes);

// Animation Speed
Text('Animated').fadeIn(duration: 300.milliSeconds);
```
</Card>

### ⏳ Time Conversions
{/*  ------------------ */}
<ResponseField name="days" type="Duration" post={["getter"]}>

Converts the integer value to a `Duration` object representing days

<CodeGroup>
    ```dart Lists icon="list" lines wrap
    final longWait = 3.days; // Duration(days: 3)
    ```
</CodeGroup>
</ResponseField>
